# Changelog

## gglinedensity (development version)

## gglinedensity 0.2.0

CRAN release: 2025-05-22

- [`stat_path_density()`](https://hrryt.github.io/gglinedensity/reference/stat_line_density.md)
  is added for binning paths connecting observations in the order in
  which they appear in the data.

- [`stat_line_density()`](https://hrryt.github.io/gglinedensity/reference/stat_line_density.md)
  now computes count, ncount and ndensity variables, and the `normalise`
  argument is therefore removed.
